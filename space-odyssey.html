Space Odyssey

Narrative game for the Chicago Refugee Coalition.

Role Narrative Designer · UI/UX Designer · Researcher
Platform Twine → Ren'Py (in development)
Collaboration CRC, Prof. Richard, DePaul University
Timeline 6 months
Audience Ages 14+
Empathy Design Client Collaboration Narrative UX Social Impact

Overview

A narrative-driven serious game created with the Chicago Refugee Coalition to build empathy around forced migration. The game is a sci-fi narrative experience designed to be emotionally grounded — accessible, respectful, and educational — playable in 15 minutes.

Design Goal

Design an emotionally grounded refugee story players can finish in 15 minutes — accessible, respectful, and educational.

My Contributions

  • Mapped full narrative branching logic using flowcharts
  • Designed UI wireframes and mockups
  • Codex screens (faction lore, character bios, political updates)
  • Transitioned design documentation from Twine → Ren'Py
  • Collaborated on tone, pacing, and worldbuilding documents
  • Integrated UX principles to support readability, flow, and emotional clarity
  • Conducted internal playtests with peers to refine logic + layout

In Progress

Active Development
  • Ren'Py build: Coding in dialogue logic and conditional events
  • Trust system: Visualizing bias shifts per interaction
  • Finalizing UI styles (color palette, typography, hover states, sound cues)

Impact

This project pushes the boundaries of what serious narrative games can do when grounded in real-world parallels. While still in development, the systems and worldbuilding work already model how to blend gameplay, UI, and sociopolitical storytelling into a cohesive experience.
